实验环境的配置

  1. 安装QEMU模拟硬件和行为 sudo apt install qemu-system-x86 （QEMU使用的是SeaBIOS）
  2. 获取jyy os课的实验代码 wget -r -np -nH --cut-dirs=2 -R "index.html*" "https://jyywiki.cn/os-demos/introduction/debug-firmware/"
  3. 获取16位x86汇编的汇编器NASM用于制作img文件 sudo apt install nasm

背景：BIOS中断

在有固件没有os的环境下，想要实现屏幕控制可以通过BIOS中断达成，具体方式位（汇编指令）int <code> ，其中code表示具体的中断类型。在执行这个指令后，BIOS将根据某些特定寄存器中的值来完成对应功能。
